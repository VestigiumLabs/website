# VestigiumLabs Website

Site officiel de **VestigiumLabs**, un laboratoire indépendant consacré à la recherche, au développement et au partage de connaissances en :

- 🔎 Investigation numérique (Digital Forensics)
- 🛡️ Cybersécurité
- ⛓️ Technologies blockchain
- 💻 Développement d'outils Open Source

L'objectif de ce dépôt est de développer le site web officiel de VestigiumLabs.
Ce laboratoire est en construction. Chaque version de ce site teflète les compétences acquises au fil du temps.

---

## Vision

Le site a pour vocation de devenir une plateforme de référence francophone où seront publiés :

- des projets Open Source ;
- des recherches techniques ;
- des articles de vulgarisation et d'analyse ;
- des retours d'expérience ;
- de la documentation.

Le développement sera progressif afin de faire évoluer le projet au rythme des compétences acquises.

---

## Objectifs

- Présenter les projets développés par VestigiumLabs
- Publier des articles techniques
- Documenter les recherches
- Centraliser les ressources
- Servir de documentation officielle des outils développés

---

## Technologies (prévision)

Le choix des technologies évoluera avec le projet.

À ce stade, les pistes envisagées sont :

- HTML5
- CSS3
- JavaScript
- GitHub
- Cloudflare Pages

Des technologies complémentaires pourront être ajoutées au fil de l'évolution du laboratoire.
